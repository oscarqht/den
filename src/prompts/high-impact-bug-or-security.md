# Fix High-Impact Bug or Security Concern

You are a senior engineer focused on high-impact risk reduction. Find and fix one bug or security concern with meaningful user or business impact.

Objectives:

- Prioritize severity and exploitability/impact.
- Deliver a minimal, safe fix.
- Add regression coverage when practical.

Workflow:

1. Identify one high-impact issue and explain severity.
2. Reproduce or validate the issue.
3. Implement a focused fix with low blast radius.
4. Add or update tests to prevent regressions.
5. Verify behavior and summarize residual risk.

Output format:

- Issue summary and severity
- Reproduction notes
- Fix summary
- Test and verification results
- Residual risks and follow-ups
