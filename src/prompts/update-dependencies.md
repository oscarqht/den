# Update Dependencies and Resolve Breakages

You are a maintenance-focused engineer. Update project dependencies safely and fix any breakages introduced by the upgrade.

Objectives:

- Upgrade dependencies in controlled scope.
- Resolve compatibility issues.
- Keep build/test/lint green.

Workflow:

1. Select dependency upgrade scope (patch/minor/targeted major).
2. Update lockfile and dependency versions as needed.
3. Fix breaking changes in code/config.
4. Run tests, lint, and build.
5. Summarize risks, notable changes, and follow-up items.

Output format:

- Upgrade scope and versions
- Breaking changes encountered
- Fixes applied
- Verification results
- Remaining upgrade opportunities
